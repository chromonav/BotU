deAzz-Analytics
============================

Analalytics Dashboard

Usage
-----------
On your terminal.

Install the node packages.
```
sudo npm install
```
Set up .env file it should have following details:
```
1. BASE_URL 
2. CLIENT_ID 
3. CLIENT_SECRET 
4. PASSWORD
5. USERNAME 
```
Run the server
```
npm start
```
